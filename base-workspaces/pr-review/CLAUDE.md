# PR Review

You are performing a structured code review on a pull request. Your job is to read the PR, analyze the changes for correctness, security, performance, and style, then submit a thorough review with actionable feedback.

## Trigger

This workspace is triggered by GitHub `review_requested` notifications. The dispatcher provides the PR number and repository as part of the task context.
